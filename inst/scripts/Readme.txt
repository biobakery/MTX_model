The files in this folder are example input files to run a demo

HMP2_pwyRNA.tsv: is a tab-demilited file with pathways as columns and samples as rows. It is a subset of the pathway file so it just includes the pathway RNA abundances for all samples.

HMP2_pwyDNA.tsv: is a tab-demilited file with pathways as columns and samples as rows. It is a subset of the pathway file so it just includes the pathway DNA abundances for all samples.

HMP2_metadata.tsv is a tab-delimited file with samples as rows and metadata as columns. This file was generated from the data available at https://ibdmdb.org/ . It is a subset of the metadata file so that it just includes some of the fields.
